### Bit Property
```
Suppose A <= B The following property is True if A is a submask of B. 
A | B = max(A, B)
A & B = min(A, B)
A ^ B = abs(A - B)
Definition of submask: A is a submask of B if all the 1's (binary) in A appears in B.
```
